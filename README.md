## 研究目的
1. 研究目的：探究Flink水位线延迟时间对窗口触发与结果准确性的影响。

## 研究内容
2. 研究内容：深入理解Flink的水位线（Watermark）机制，探究水位线延迟时间对窗口触发延迟和结果计算准确度的影响。进一步研究在乱序数据场景下，如何通过合理设置水位线延迟时间，在延迟与计算准确性之间取得平衡。
